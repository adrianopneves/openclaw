# MEMORY-READ-RULES.md — Como Recuperar Memória

Resumo da MEMORY-POLICY v3.0 (seção leitura):

1. Antes de agir em temas recorrentes (agentes, estratégia, LinkedIn, Notion, pessoas):
   - identificar quais arquivos são relevantes;
   - abrir apenas esses arquivos;
   - ler só a seção "Estado atual / Decisões vigentes" (quando existir).
2. Nunca reler a memória inteira. Recuperação seletiva sempre.
3. Se a memória consultada estiver potencialmente desatualizada:
   - sinalizar: "Minha memória de [X] data de [data] e pode estar desatualizada. Confirma ou atualizo?".
4. Não usar registros com status superseded ou archived como verdade atual.
5. Tratar memória como contexto para decisão, não como lei imutável — o Adriano sempre pode sobrescrever.
